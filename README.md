# GPT бот-репетитор

Телеграм бот, распознающий речь.


## Описание

Бот распознаёт речь пользователя и отправляет его
в формате текстового сообщения. Функциональность свою он осуществляет 
с помощью SpeechKit от Яндекс.

## Инструкции по использованию
- Начало работы:
  - Найди бота в Telegram: t.me/medokok_bot.
  - Запусти бота командой /start.
  - Ознакомься с инструкцией командой /help.
- Распознавание речи:
  - Для начала работы введите /stt.
  - Отправьте голосовое сообщение, которое необходимо обработать.
  - Получите тест сообщения.
- Распознать можно 3 сообщения длиной не более 15 секунд.
- Бот отправит логи по команде /debug, скрытой от масс.
